Design Pattern Questions

What is a Strategy Pattern?

The strategy pattern extracts algorithms from a certain class and makes a different class for every single algorithm.

What is an Observer Pattern?

Observer Pattern lets several observer objects be notified when a subject is changed in come way.

What is a Composite Pattern?

Composite Pattern composes objects into tree structures to represent part-whole hierarchies.

What is a Decorator Pattern?

Decorator pattern allows a user to add new functionality to an existing object without altering its structure.

What is a Facade Pattern?

Facade pattern hides the complexities of the system and provides an interface to the client using which the client can access the system.
